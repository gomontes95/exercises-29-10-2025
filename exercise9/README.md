Gestión de Tareas Simple (CRUD): Crea un programa que permita al usuario realizar operaciones básicas de gestión de tareas:

a. Crear una nueva tarea.
b. Reportar (listar) todas las tareas.
c. Update (actualizar) una tarea existente.
d. Delete (eliminar) una tarea. 

Puedes usar una lista o un diccionario para almacenar las tareas.
